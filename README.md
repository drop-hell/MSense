# MSense
  
**Official implementation/deployment** of the paper:  
MSense: A Multimodal Spatio-Temporal Network with Selective Embeddings


## Requirements

This project requires the following Python libraries:

- python==3.8.8
- numpy
- pandas==1.2.5
- pytorch==1.8.1
- scikit-learn==1.3.2
- tslearn==0.6.2
- sktime
